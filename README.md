## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/omarcosvbn/meteora.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the API
   ```sh
   cd src/app
   json-server db.json --port 3001
   ```
4. Run the project
   ```sh
   npm run start
   ```

The project can be visualized at [localhost](http://localhost:3000/).
